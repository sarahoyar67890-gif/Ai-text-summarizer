# 🧠 AI Text Summarizer

A simple and powerful **AI-based Text Summarization Web App** built using **Hugging Face Transformers** and **Streamlit**.
This project takes long articles as input and generates concise summaries automatically.

---

## 🌸 Project Overview

This application uses a pre-trained NLP model to:

* Understand long text
* Extract key information
* Generate a shorter, meaningful summary

It is useful for:

* Students 📚
* Researchers 🔬
* Content creators ✍️
* Anyone who wants quick insights from long articles

---

## 🚀 Features

✨ Summarizes long text instantly
✨ Clean and simple UI using Streamlit
✨ Uses state-of-the-art NLP model
✨ Easy to run locally

---

## 🛠️ Tech Stack

* Python 🐍
* Hugging Face Transformers 🤗
* PyTorch 🔥
* Streamlit 🌐
* Pandas 📊
